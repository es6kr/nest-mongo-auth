# Nest Mongo Auth

[![npm](https://img.shields.io/npm/l/@nestjs/core.svg?sanitize=true)](https://opensource.org/licenses/MIT)

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript template with Mongoose and Passport

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
